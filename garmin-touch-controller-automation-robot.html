Garmin Touch Controller - Automation Robot

People assume that the Ender 3 Creality board is only good for 3d printing applications, but I had more interesting ideas on what to do with the platform. The goal of this project was to convert the normal functionality of an ender 3 printer into testing automation device. Some background information is needed here. Large aviation companies use Garmin Touch Controllers in their jets in order to provide the pilot information about fuel, aerlerons, etc. Engineers need to perform multiple hours of regression testing which involves them pushing the buttons on these controllers and marking either a pass or a fail. This is needed each time a new software drops or hardware is changed which is quite often. As you can image this process is highly tedious and involves hundreds of hours. I found a better way to automate this process
